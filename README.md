## Development

Clone the repository

		git clone git@github.com:rubygems/adoption-center.git

Install bundled gems

		bundle install

Copy over the database file

		cp config/database.example.yml config/database.yml

Setup the database

		bundle exec rake db:setup

Start up rails!

		rails server

Run the tests!

		rake test
